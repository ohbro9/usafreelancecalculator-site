# USA Freelance Calculator Toolkit Brand Alignment Checklist

## Status
- Saved on: 2026-03-27 UTC
- Purpose: Lock the brand-alignment strategy so future polish work does not drift or mix up priorities.
- Scope: Deploy-ready toolkit docs mirror

## Core Rule
- Same brand DNA, not the same face.
- Keep suite behavior consistent.
- Keep each tool's visual personality distinct.
- Prefer light alignment over redesign.

## Must Align Now
- Top toolkit navigation behavior
- Primary / secondary / ghost button hierarchy
- Back / Next action philosophy
- Export action order and naming
- Trust / utility chip styling pattern
- Footer / disclaimer tone
- Spacing rhythm between sections and cards
- Border-radius family
- Toast / modal language and interaction tone
- Focus and active-state accessibility behavior

## Good To Align Later
- Shared icon-family direction
- Motion timing and transition grammar
- Heading / eyebrow / supporting-copy rhythm
- Empty / warning / helper-state wording
- Shared OG / social preview system
- Shared helper utilities for duplicated UI behavior

## Never Redesign
- Hourly Rate's bright, approachable SaaS tone
- Platform Fee's comparison-led product feel
- Tax Estimator's dark authority-finance tone
- Income Goal Planner's luxury-planner mood
- Budget Planner's executive black-gold control tone
- Each tool's core result layout
- Each tool's hero mood
- Each tool's tool-specific microcopy voice

## Exact Per-Tool Checklist

### Hourly Rate
- Change: align nav, CTA hierarchy, export block, footer tone, trust-chip family.
- Preserve: bright SaaS energy, reverse planning mode, fast low-friction flow.
- Do not touch: approachable first impression and reverse-mode identity.

### Platform Fee
- Change: align nav behavior, next-tool action grammar, export naming, footer/disclaimer tone.
- Preserve: comparison-led layout, trust-chip framing, Client Pays vs You Keep logic.
- Do not touch: comparison-first product identity.

### Tax Estimator
- Change: align nav rhythm, helper/export messages, CTA hierarchy, footer tone.
- Preserve: dark authority feel, serious finance-tool layout, careful microcopy.
- Do not touch: high-trust structure and finance authority tone.

### Income Goal Planner
- Change: align nav rhythm, Back / Next grammar, export block, footer/disclaimer system.
- Preserve: premium planner styling, sticky summary, ambition/planning mood.
- Do not touch: luxury-planner personality and sticky product feel.

### Budget Planner
- Change: align nav logic, export naming, helper/toast language, footer tone.
- Preserve: black-gold executive tone, final-report identity, profile preset cards.
- Do not touch: final-sink premium personality and dashboard-control mood.

## Implementation Order
1. Nav behavior and active-state polish
2. Button hierarchy and CTA wording
3. Export action consistency
4. Footer and disclaimer system
5. Spacing and radius discipline
6. Toast / modal language cleanup
7. Trust-chip family cleanup

## Working Rule
- If a change improves consistency but weakens a tool's core personality, reject it.
- If a change preserves personality and improves suite cohesion, keep it.
- Prioritize behavior consistency over visual sameness.
