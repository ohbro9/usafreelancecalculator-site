Privacy

Tool usage is designed to stay lightweight and browser-local.

This toolkit is built as a static website. The main calculators work in your browser, and connected handoff between tools is stored locally in your own browser storage so you can move through the workflow more easily.

Storage
Local browser storage is used for calculator state and toolkit handoff
Hosting
The website is statically hosted and does not require sign-up to use the tools
Exports
Reports are generated in the browser when available

What may be stored in your browser

  • Current tool inputs and preferences for convenience.
  • Connected toolkit handoff data when you move from one tool to the next.
  • Saved settings or presets where a tool explicitly offers that behavior.

How exports work

PDF, image, and similar report actions are generated from the page you are using. Some export features depend on third-party browser libraries loaded from public CDNs. If those libraries are blocked by your network, extensions, or browser settings, some export options may be unavailable.

Clearing saved state

You can reset individual tools with each tool's reset action. Connected toolkit flow can also be cleared with the toolkit reset or start-over actions where available. If you want a full clean slate, you can also clear site data for this website in your browser settings.

Practical privacy note

This page describes how the toolkit is intended to behave today. Website features may evolve over time, and this privacy page may be updated as the public website matures.
